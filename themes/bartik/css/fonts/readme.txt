This directory contains local font files for the Muthoot Finance website.
These fonts were previously loaded from external CDN URLs but are now stored locally for better performance and reliability.

Font files included:
- FrutigerBlack (eot, woff2, woff, ttf, svg)
- FrutigerLTStd55RomanRegular (eot, woff2, woff, ttf, svg)
- frutigeb (eot, woff2, woff, ttf, svg)
